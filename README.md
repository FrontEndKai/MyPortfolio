## ğŸš€ Deployment

### Deploy to Vercel (Recommended)

1. Install Vercel CLI
```bash
npm install -g vercel
```

2. Deploy
```bash
vercel
```

Or push to GitHub and import to Vercel dashboard.

### Deploy to Netlify

1. Build the project
```bash
npm run build
```

2. Drag and drop the `dist` folder to Netlify

Or use Netlify CLI:
```bash
npm install -g netlify-cli
netlify deploy --prod
```

### Deploy to GitHub Pages

1. Install gh-pages
```bash
npm install --save-dev gh-pages
```

2. Add to package.json scripts:
```json
"predeploy": "npm run build",
"deploy": "gh-pages -d dist"
```

3. Deploy
```bash
npm run deploy
```

## ğŸ“ Customization

### Update Your Information

1. **Home Page** - Edit `src/pages/Home.jsx`
   - Change name, title, description
   - Update social media links

2. **About Page** - Edit `src/pages/About.jsx`
   - Add your skills
   - Update education/experience

3. **Projects Page** - Edit `src/pages/Projects.jsx`
   - Add your projects with descriptions
   - Include live links and GitHub repos

4. **Contact Page** - Edit `src/pages/Contact.jsx`
   - Update email, phone, location
   - Configure contact form

### Change Colors

Edit `src/styles/index.css` - Look for `:root` variables:
```css
--primary-color: #3b82f6;
--secondary-color: #1e40af;
```
